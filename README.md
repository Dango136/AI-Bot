This is my first Telegram bot made with Python. This project uses AI for one of it's functions, which lets it detect the person on the photo.
Made by Dango136.
Functions :
Hello - The bot says hello to you!
Hehe - The bot will say "Hehe" and multiple it on the number you say it.
Mini app - Opens a blank mini app.
Message - copies and tells back the message you typed.
Photo - Send a photo and the bot will detect the person on it.
